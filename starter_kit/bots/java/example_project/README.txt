An example directory structure for folder containing Pirate Bots in Java.

This sample code shows how to include 2 files in a code project. The 2 files are under "bots" package
 * A main MyBot.java containing the MyBot class which implements PirateGame (This file be under the "bots" package).
 * An utility class MyBotUtils, called from MyBot

To run:
run.bat bots/java/example_project bots/python/NothingBot.py
or
run.bat bots/java/example_project bots/python/NothingBot.py maps/default_map.map
