An example directory structure for folder containing Pirate Bots in C#.

This sample code shows how to include 2 files in a code project.
 * An entry point MyBot.cs containing a class which implements PirateGame.
 * An utility class MyBotUtils, called from MyBot

To run:
run.bat bots/csharp/example_project bots/python/NothingBot.py
or
run.bat bots/csharp/example_project bots/python/NothingBot.py maps/default_map.map
