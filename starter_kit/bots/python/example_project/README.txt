An example directory structure for folder containing Pirate Bots in python.

This sample code shows how to include 2 files in a code project:
 * A main my_bot.py file containing a do_turn function (this filename is mandatory)
 * A utility module my_bot_utils, called from my_bot.

To run:
run.bat bots/python/example_project bots/python/example_bots/nothing_bot.py
or
run.bat bots/python/example_project bots/python/example_bots/nothing_bot.py maps/default_map.map
